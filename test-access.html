📋 System Overview

The access code system uses a rotating loop structure with 3 codes:

  • User 1: IMU9846
  • User 2: IMU350
  • User 3: IMU456

🔄 How the Loop Works

Step 1: User 1 enters IMU9846 → Gets access ✅
→ System automatically moves to User 2
Step 2: User 2 enters IMU350 → Gets access ✅
→ System automatically moves to User 3
Step 3: User 3 enters IMU456 → Gets access ✅
→ System loops back to User 1
Step 4: User 4 enters IMU9846 → Gets access ✅
→ And the cycle continues...

🎯 Key Features

  • ✅ Only ONE code works at a time
  • ✅ After successful login, code automatically rotates to next
  • ✅ Loops back to first code after last code is used
  • ✅ Each user session is tracked separately
  • ✅ Code rotation is shared across all users (stored in localStorage)
